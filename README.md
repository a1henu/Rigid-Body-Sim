# Rigid-Body-Sim
Taichi implementation for lab 1 in the course PhySim4CG at Peking University

## Environment

This project is configured for `uv` with Python `3.11`.

Initial setup:

```bash
uv sync
```

Run the app:

```bash
uv run python main.py --demo single_body --steps 0
```

Run tests:

```bash
uv run pytest -q
```
